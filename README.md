# Resume PDF Generator

This repository contains a script to generate a professional PDF version of your resume directly in the current working directory.

## How It Works

Simply run the script, and it will automatically create a formatted PDF of your resume in the same folder.

## Usage

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd your-repo
   ```

2. Run the script:

   ```bash
   python resume_generator.py
   ```

3. A PDF file named `resume.pdf` will be generated in the current directory.

## Requirements

* [Python 3.x](https://www.python.org/downloads/)
* Required dependencies (install via `pip`):

  ```bash
  pip install -r requirements.txt
  ```

## Output

The script will generate a polished, ready-to-use resume in PDF format, saved as:

```
./resume.pdf
```
